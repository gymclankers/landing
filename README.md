# GymClankers

AI-forged workout anthems. Dark industrial gym music, generated by machines.

**[gymclankers.com](https://gymclankers.com)**

## What This Is

Static landing page for the GymClankers music project. Single-page hero with animated wordmark, Spotify embed, and social links.

## Tech

- Pure HTML/CSS/JS — no build step, no framework
- Cloudflare Pages (auto-deploys on push to `main`)
- Progressive background loading (base64 placeholder → full image)
- SVG wordmark with electricity trace animation + spark particles
- Google Analytics (GA4)

## Structure

```
index.html          Landing page
css/style.css       Styling + animations
js/electric.js      Spark particles + letter flicker
img/
  banner_01.webp    Hero background (WebP, 950 KB)
  og-banner.webp    Social preview card
  favicon.svg       Robot arm + dumbbell
```

## Deploy

Push to `main`. Cloudflare Pages picks it up automatically.

```
git push origin main
```
